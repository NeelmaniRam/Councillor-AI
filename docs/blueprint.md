# **App Name**: IvyVoice Career Guide

## Core Features:

- Voice Input & STT: Accept voice input from the user and convert it to text using Speech-to-Text technology.
- AI-Driven Conversation: Conduct a guided conversation with the student, asking questions about their interests, strengths, and preferences, powered by an LLM.
- Real-time Insight Extraction: As the conversation progresses, extract key insights (interests, strengths, constraints) and update the side panel dynamically, working as a tool.
- Career Path Hypothesis Testing: Propose a set of career clusters and test them against the student's preferences to narrow down the best-fit paths.
- Career Discovery Report: Generate a structured report summarizing the student's profile, top interests, key strengths, and recommended career paths.
- Downloadable PDF Report: Allow users to download the Career Discovery Report in PDF format.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), to create a calming and trustworthy atmosphere suitable for a guidance app.
- Background color: Light blue (#E3F2FD), almost white, offering a gentle and unobtrusive backdrop.
- Accent color: A subdued purple (#9575CD) to highlight important insights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look. Note: currently only Google Fonts are supported.
- Use simple, clear icons to represent different interests, strengths, and career paths.
- A clean, dashboard-style layout with a large microphone button and a side panel for real-time insight display.
- Subtle waveform animation during voice input and AI processing to provide a sense of activity and engagement.
